## 简述

一个实现 Stack 机制的逻辑组件

## 场景

### 复杂带分叉的向导页面

向导一般由多个页面组成, 甚至在第 3 个页面, 可以选择跳到第 4 页 或 第 6 页。甚至重新回到第 2 页
无论有多复杂, 本质上就是一个 stack 机制

## Install

npm install --save react-logic-stack
